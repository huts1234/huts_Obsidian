## 一、Markdown 核心语法

### 1. 基础元素
| **语法**                | **效果**                     | **示例**                      |
|-------------------------|-----------------------------|-------------------------------|
| `# 标题`                 | 一级标题                     | `# 一级标题` → <h1>一级标题</h1> |
| `## 标题`                | 二级标题                     | `## 二级标题` → <h2>二级标题</h2> |
| `**加粗**`               | 加粗文本                     | `**加粗**` → **加粗**          |
| `*斜体*`                 | 斜体文本                     | `*斜体*` → *斜体*              |
| `~~删除线~~`             | 删除线                       | `~~删除线~~` → ~~删除线~~       |
| `[链接](https://...)`    | 超链接                       | `[Google](https://google.com)` → [Google](https://google.com) |
| `![图片](image.jpg)`     | 插入图片                     | `![Logo](logo.png)` → 显示图片   |
| `> 引用`                 | 引用块                       | `> 名人名言` → <blockquote>名人名言</blockquote> |
| `` `代码` ``             | 行内代码                     | `` `print("Hello")` `` → `print("Hello")` |
| ``` 代码块 ```           | 多行代码（指定语言）           | <pre><code>print("Hello")</code></pre> |

### 2. 列表与表格
#### 无序列表
```markdown
- 项目1
  - 子项目1
  - 子项目2
- 项目2
```
#### 有序列表
```markdown
1. 第一项
2. 第二项
3. 第三项
```
#### 表格
```markdown
| 列1   | 列2   | 列3   |
|-------|-------|-------|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |
```
效果：

| 列1   | 列2   | 列3   |
|-------|-------|-------|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |

## 二、扩展语法（GFM / CommonMark）

### 1. GitHub Flavored Markdown (GFM)
| **功能**          | **语法**                     | **示例**                      |
|-------------------|-----------------------------|-------------------------------|
| 任务列表          | `- [x] 完成`                 | ✅ 完成                        |
| 表格对齐          | `:---:` 居中，`---:` 右对齐  | 见上方表格示例                  |
| 代码块高亮        | ```python 代码块 ```         | 语法高亮显示                    |
| 自动链接          | `<https://example.com>`      | <https://example.com>         |

### 2. 其他扩展功能
- **脚注**：
  ```markdown
  这是一句话[^1]需要注释。
  [^1]: 这里是脚注内容。
  ```
- **定义列表**：
  ```markdown
  术语1
  : 定义1
  术语2
  : 定义2
  ```
- **数学公式（KaTeX）**：
  ```markdown
  行内公式：$E=mc^2$  
  块级公式：
  $$
  \sum_{i=1}^n i = \frac{n(n+1)}{2}
  $$
  ```

## 三、实用技巧与工具

### 1. 效率工具推荐
| **工具**         | **功能**                     | **平台**                     |
|------------------|-----------------------------|-----------------------------|
| VS Code          | 实时预览 + 插件扩展           | Windows/macOS/Linux         |
| Typora           | 所见即所得编辑器              | Windows/macOS/Linux         |
| Obsidian         | 知识库管理 + 双向链接          | 全平台                       |
| Markdown All in One| 快捷键增强（VS Code插件）      | VS Code 插件                 |

### 2. 格式转换工具
- **Pandoc**：支持 Markdown 转 PDF/Word/HTML 等格式  
  ```bash
  pandoc input.md -o output.pdf
  ```
- **Markdown to Slides**：生成幻灯片  
  ```bash
  npm install -g reveal-md
  reveal-md slides.md --theme=night
  ```

## 五、常见问题与解答

### 1. 如何转义特殊字符？
使用反斜杠 `\` 转义：
```markdown
\*这不是斜体\* → *这不是斜体*
```

### 2. 如何实现多级标题自动编号？
需借助插件（如 Typora 的「插入目录」功能）或生成工具：
```markdown
<!-- 生成目录（部分工具支持） -->
[TOC]
```

### 3. Markdown 文件如何导出为 PDF？
- **VS Code**：安装 `Markdown PDF` 插件  
- **Typora**：直接菜单导出（文件 → 导出 → PDF）
