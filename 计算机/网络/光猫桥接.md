# 光猫桥接详细指南

---

## 一、光猫桥接的核心价值
**光猫桥接模式**是将光猫从"路由模式"切换为"透明传输模式"，使其仅承担**光电信号转换**功能，而将拨号、路由、NAT等任务移交专业路由器完成。其核心优势包括：

| **对比维度** | **桥接模式**                | **路由模式**             |
| -------- | ----------------------- | -------------------- |
| 网络性能     | 降低延迟（减少NAT层级），提升带宽利用率   | 多一层NAT转换，性能损耗约10-15% |
| 设备控制权    | 可自定义路由器功能（DDNS/VPN/QoS） | 受限于光猫固件功能            |
| 稳定性      | 避免光猫性能瓶颈（如连接数限制）        | 高负载易导致光猫死机           |
| 适用场景     | 游戏/直播/NAS用户             | 普通上网需求               |

## 三、桥接设置分步教程
### 1. 登录光猫管理界面
1. 连接光猫LAN口与电脑，设置电脑IP为**自动获取**  
2. 浏览器访问光猫管理地址（常见于设备底部标签）：  
   - 电信：`http://192.168.1.1`  
   - 联通：`http://192.168.1.1/cu.html`  
   - 移动：`http://192.168.1.1`  
3. 使用**超管账号**登录（非普通user账号）

### 2. 删除原有拨号连接
1. 进入**网络 → 宽带设置**  
2. 找到`INTERNET`或`4_INTERNET`连接，记录**VLAN ID**（如41）  
3. 删除该连接（点击"删除"或"移除"）

### 3. 新建桥接连接
| **参数**         | **设置值**                          | **说明**                     |
|------------------|-------------------------------------|-----------------------------|
| 连接模式         | 桥接                                | 必选                         |
| 业务类型         | Internet                            | 部分光猫需选择               |
| VLAN模式         | 启用                                | 必须与之前记录的VLAN ID一致  |
| VLAN ID          | 填入之前记录的值（如41）             | 不同地区不同，不可随意修改   |
| 绑定端口         | LAN1（或根据需要选择）               | 建议仅绑定路由器连接的端口   |

**操作路径**（以电信天翼网关为例）：  
网络 → 网络设置 → 新建WAN连接 → 按上表填写 → 保存

### 4. 关闭光猫路由功能
- **DHCP服务**：网络 → DHCP设置 → 关闭  
- **WiFi功能**：无线网络 → 关闭SSID（避免信号干扰）  

### 5. 配置路由器拨号
1. 将路由器WAN口连接至光猫LAN1  
2. 登录路由器管理界面（如`192.168.50.1`）  
3. 选择**PPPoE拨号**，输入宽带账号密码  
4. 开启IPv6（如需）：选择**Native**或**Passthrough**模式

## 四、桥接后优化设置
### 1. 路由器推荐配置
| **功能**       | **建议值**                      | **作用**                     |
|----------------|---------------------------------|-----------------------------|
| MTU            | 1492（PPPoE默认）              | 避免数据包分片降低效率        |
| NAT类型        | Full Cone                      | 提升P2P连接成功率（游戏/NAS） |
| QoS            | 按设备分配带宽                  | 保障关键应用（视频/游戏）      |
| 防火墙         | 开启SPI和DoS防护               | 平衡安全与性能                |

### 2. 测速与验证
- **测速平台**：Speedtest（选择同运营商节点）  
- **延迟测试**：  
  ```bash
  ping 223.5.5.5 -n 20  # 阿里云DNS
  tracert www.baidu.com  # 查看路由跳数
  ```

## 五、常见问题解决
### 1. 桥接后无法上网
- **检查VLAN ID**：必须与运营商一致（错误则数据无法识别）  
- **确认账号密码**：大小写敏感，建议复制粘贴  
- **重启顺序**：光猫→路由器→终端设备  

### 2. IPTV无法使用
- **单独绑定端口**：在光猫中为IPTV业务新建一条**路由模式**连接，绑定到LAN2口  
- **组播设置**：开启路由器**IGMP Proxy**（OpenWRT路径：网络→接口→LAN→物理设置→勾选"IGMP嗅探"）  

### 3. 运营商远程恢复
- **禁用TR069**：在光猫"远程管理"页面禁用TR069连接（防止自动恢复路由模式）  
- **物理隔离**：拔掉光猫电话线（部分光猫通过电话线接收配置更新）
